Restaurant Forum

I created this app to give users the ability to choose a state and city and see what people are saying about restaurants in that city. Users can add and delete cities and restaurants. Comments can be added to restaurants like name, phone, address, and overall information about the restaurant.

The APP and API were created from scratch using React for the client-side and PostgreSQL and Node for the back end.

Technology used:

  • React
  • CSS
  • JavaScript
  • Knex
  • PostgresQL
  • Node
  • Express
  • HTML
  • CSS

Virtual Partner

I created this app to give users the ability to look up exercises for their workout. It allows them to search for a specific exercise or a body group. In either option, they are provided with a description and three Youtube videos to further their understanding of the exercise of choice.

The API comes from both "Youtube" and "wger REST API" for the exercises. The user is given all the exercises in the database that are in English and are complete in their format. I believe that for someone looking for a quick lookup on the go, the app provides it for them.

Technology used:

  • HTML
  • CSS
  • JavaScript
  • jQuery
